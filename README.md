# <sup>I</sup>NST<sup>A</sup>L<sup>L</sup>

Install packages with `npm`, `yarn`, or `pnpm` based on the lock file present in the repo.

## Installation

```sh
npm i -g nstl [yarn] [pnpm]
```
## Usage

```sh
nstl [package-1] [package-2]
```
